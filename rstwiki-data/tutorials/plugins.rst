Overview
--------

There are 3 layers you CAN extend :

* Quick-XApp
* XApp-Studio
* XApp-Mobile-Core

Each layer is built or inherited on each other :

.. image :: http://www.mydiagram.org/xapp-diagrams/poem/model/23/svg


XApp-Studio provides a coding temlate for all layers.
